# urule-springboot
基于开源规则引擎URULE的DEMO


[https://github.com/youseries/urule](https://github.com/youseries/urule)

[文档视频教程地址](https://github.com/youseries/urule)

## 启动

> ```bash
> $ mvn spring-boot:run
> ```

## 配置
* D盘新建缓存目录 d:/repo (位置可以在配置文件里修改)
* 根据官方文档配置规则（配置页：http://localhost:8080/urule/frame）
